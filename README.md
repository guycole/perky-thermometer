# perky-thermometer
BeagleBone Black using the DHT20 thermometer/humidity sensor.


## Pins
    | Header | Pin | Name      | Description |
    |--------|-----|-----------|-------------|
    | P9     |  1  | DGND      | DHT20 Pin 3 |
    | P9     |  4  | VDD_3V3   | DHT20 Pin 1 |
    | P9     | 19  | I2C2_SCL  | DHT20 Pin 4 |
    | P9     | 20  | I2C2_SDA  | DHT20 Pin 2 |
